# Daisy Surana Portfolio

Next.js App Router version of the portfolio site.

Logo asset note: add the final Daisy logo at `/public/assets/daisy-logo.png`.

## Run locally

```bash
npm install
npm run dev
```

`npm run dev` now clears the development cache automatically before starting, so stale chunk errors from previous sessions should not recur.

## Build

```bash
npm run build
```

`npm run build` clears the production build cache first.

## Reset caches manually

```bash
npm run clean
```
